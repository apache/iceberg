## What's New in This Release

Apache Iceberg continues to evolve with exciting new features and improvements. In this example post, we'll explore some of the key highlights:

### Performance Improvements

We've made significant improvements to query performance, including:

- **Faster metadata operations**: Reduced metadata read times by 40%
- **Optimized file pruning**: Smarter partition and file filtering
- **Improved caching**: Better cache utilization for repeated queries

### New Features

Several new features have been added:

1. **Enhanced table evolution**: Support for more complex schema changes
2. **Better compaction strategies**: New options for optimizing file layouts
3. **Improved monitoring**: More detailed metrics and observability

## Getting Started

To try out these new features, simply update your Iceberg dependency:

```xml
<dependency>
    <groupId>org.apache.iceberg</groupId>
    <artifactId>iceberg-core</artifactId>
    <version>1.10.1</version>
</dependency>
```
